new <async> LED( options )

Description
The LED class allows for control of Light Emitting Diodes
Parameters
Name Type Description
options object Pin identifier or IO Options (See Instantiating a Device)
Name Type Attributes Default Description
pwm boolean <optional>
 false True if the device is wired to a PWM pin
sink boolean <optional>
 false True if the device is wired for sink drive
Examples

Using a pin number

import LED from "j5e/led";
			
			const led = await new LED(12);
			led.on();

Using a pin identifier

import LED from "j5e/led";
			
			const led = await new LED("A1");
			led.on();

Using an IO options object

import LED from "j5e/led";
			
			const led = await new LED({
			  pin: 12,
			  pwm: true
			});
			led.on();

Description
stop Stop the led from blinking, pulsing, fading, or animating
Returns
Examples
Pulse an LED and then stop after five seconds

			import LED from "j5e/led";
			
			const led = await new LED({
			  pin: 12,
			  pwm: true
			});
			led.pulse(500);
			
			// Stop pulsing after five seconds
			timer.setTimeout(function() {

			}, 5000);
